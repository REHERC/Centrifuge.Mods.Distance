{Prerequisites}

INSTALLATION

To install this mod navigate to your local distance installation folder and open Distance_Data.
If you bought the game on Steam you can right-click the game in your library then go to Properties, then in the Local Files tab click on the button labeled Browse Local Files. From there, you can open the Distance_Data folder mentioned above.

Once in the Distance_Data folder, copy all of the folders from the mod zip file inside Distance_Data and replace files if asked. Copying this readme file isn't required but is advised if you need quick documentation in the future.
