USING THE MOD

The mod adds a new options menu in the Centrifuge settings called Editor Additions, this is where you can enable features of the emod.

  • Enable Dev Objects Folder Enables the Dev folder in the level editor Library tab.
  • Advanced Music Selection Display hidden dev music cues in the level editor music selector window.
  • List Workshop Levels Allows to open the workshop levels from the level editor (read only).
