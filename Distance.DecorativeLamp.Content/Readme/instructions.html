USING THE MOD

The mod adds a new options menu in the Centrifuge settings called Decorative Lamp, this is where you can configure your Lamp.

  • Lamp Enabled Adds a lamp to the car 3D model when checked.
  • 3D Model Select the lamp 3D mesh.
  • Spinning Lamp Makes the lamp model spin.
  • Lamp Spin Speed Sets the speed at which the lamp rotates.
  • Lamp Scale Set the lamp model size.
  • Light Intensity Set the lamp brightness.
  • Light Range Set the lamp light distance.
  • Flare Brightness Set the lens flare brightness.
