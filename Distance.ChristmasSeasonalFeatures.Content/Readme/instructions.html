USING THE MOD

The mod adds a new options menu in the Centrifuge settings called Christmas Features, this is where you can configure the mod.

  • Activation Mode Select when the mod should be enabled.
  • Time Zone Format Select between local or UTC time zone.
    This setting is used in date calculations for some activation modes.
  • Select Features Select which features are active when the mod is enabled.
    • Modify Loading Screen Backgrounds Display Christmas themed loading screens.
      (does not apply for Lost to Echoes loading screens)
    • Decorative Snow Display snow on the car screen.
      (only applies for vehicles having a screen on them)
    • Decorative Reindeer Hat Add a cosmetic reindeer hat to the car.
      (may not apply on all cars)
