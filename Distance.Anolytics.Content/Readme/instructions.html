USING THE MOD

The mod adds a new options menu in the Centrifuge settings called Anolytics, this is where you can modify (and block) analytics.

  • Disable Analytics Disables connection with Google analytics services.
  • Log Analytics Activity Outputs analytics events to the console.
  • Disable Playtesting Log File Blocks file write attempts to "playtesting_data.txt".
  • Delete Playtest Log File Removes the playtesting log file from the disk.
