USING THE MOD

The mod adds a new options menu in the Centrifuge settings called Scene Dumper, this is where you can modify key bindings for the dump commands.

  • Basic Dump Key Bind Set the keyboard shortcut used to make a basic dump. (default: "LeftControl+F7")
  • Detailed Dump Key Bind Set the keyboard shortcut used to make a detailed dump. (default: "LeftControl+F8")
  • Open Dumps Folder Opens the folder containing dump logs.
