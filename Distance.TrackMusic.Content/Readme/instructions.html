USING THE MOD

The following steps all require to be done in the level editor.

IMPORTING MUSIC

Go in the Edit menu and select Add Music Track. A new object should be added to the level.

Once the object is added, select it and open the Properties tab. There, you will find the import panel.

The import panel features 5 important elements:

  • Name The name used to identify and select the track.
  • Type The file format of the current music track.
  • Embed File The file that gets played (click the yellow square on the left to import a file).
  • Download URL A direct link to an online audio file.
  • Error Displays error messages.
NOTE: using embedded files alows to play the music without an internet connection but makes the level file bigger. To import a file, use either Embed File or Download URL, not both!

EMBEDDING A FILE

After clicking the import button, a new window should appear where you can select the file.

REMOVING AN EMBEDDED FILE

To remove an embedded file, delete the corresponding Music Track object from the level.

USING A CUSTOM MUSIC TRACK

To use a Music Track open the Library tab and type music in the search bar. Look for an object called MusicTrigger with a green cube icon and place it.

In the Properties tab, click on Set Music Choice, this will add a new component on the object called Music Choice.

In the Music Choice component set the Trigger Track to the Name of the Music Track of your choice.

You can also set the level's default music by opening the Level Settings tab and selecting Set Music Choice at the bottom of the page. From there do the same as you would for a MusicTrigger.
