INSTALLING CAR MODELS

To install a car model, copy the corresponding file in either one of the following folders (you can create subfolders if you want):

  • Distance_Data/Centrifuge/Mods/{BuildPath}/Assets
  • Documents/My Games/Distance/CustomCars

Video tutorials made by ΩPUS here.

CREATING CAR MODELS

The .unitypackage file containing the development kit can be located in Distance_Data/Centrifuge/Mods/{BuildPath}/Tools/Tools.unitypackage

Official tutorial (by Nico/larnin): CustomCar wiki (github).
Keep in mind this mod mentions Spectrum because it was made before this version; this shouldn't be a problem and the same steps should also apply here.
